Built-in Scanner Functions

The additional functions provided in PocketC for Scanners generally correspond to one or more API provided by Symbol. For further details about each function, see the explanation provided in the Symbol SDK documentation.

Event System
Function Name Description
event(int time) this function in PocketC for Scanners is the same as event() in PocketC, but adds three additional event codes: [20] scanDecodeEvent, [21] scanTriggerEvent, [22] scanBatteryErrorEvent. When the application receives a scanDecodeEvent, a scan has taken place, and its data can be retrieved by calling scanGetData(). When the application received a scanTriggerEvent, the scanner's trigger has been pressed – the application does not need to respond. When the application receives a scanBatteryErrorEvent, it must inform the user of the low-battery condition.


Scanner Command Functions
Function Name Description
scanOpen() calls ScanOpenDecoder().
scanClose() calls ScanCloseDecoder().
scanAim(int bAimOn) calls ScanCmdAimOn() if bAimOn is non-zero; calls ScanCmdAimOff() otherwise.
scanGetAim() calls ScanGetAimMode().
scanLed(int bLedOn) calls ScanCmdLedOn() if bLedOn is non-zero; calls ScanCmdLedOff() otherwise.
scanGetLed() calls ScanGetLedState().
scanEnable(int bEnabled) calls ScanCmdScanEnable() if bEnabled is non-zero; calls ScanCmdScanDisable() otherwise.
scanGetEnabled() calls ScanGetScanEnabled()
scanDecode(int bStart) calls ScanCmdStartDecode() if bStart is non-zero; calls ScanCmdStopDecode() otherwise.
scanGetParams() returns a string of all the current parameters. See ScanCmdGetAllParams() in SDK documentation for details.
scanGetVersion(int type) returns a string containing the current version. When type=0, calls ScanGetDecoderVersion(); when type=1, calls ScanGetScanManagerVersion(); when type=2, calls ScanGetScanPortDriverVersion();
scanBeep(int beepType) calls ScanCmdBeep(). The beep types are defined in scanner.h, beginning with  BEEP_
scanDefaults() calls ScanCmdParamDefaults().
scanSendParams(int beepType) calls ScanCmdSendParams(). The beep types are defined in scanner.h, beginning with BEEP_
scanGetData() calls ScanGetDecodedData(), returning the decoded data. This should be called as a response to the scanDecodeEvent event.
scanGetDataType() returns the type of the previously decoded data from the previous call to scanGetData(). The result is undefined if scanGetData() is not called first. The return value is one of the values defined in scanner.h beginning with BCTYPE_


Scanner Hardware Functions
Function Name Description
scanIsSymbol() calls ScanIsPalmSymbolUnit().
scanAimDur(int dur) calls ScanSetAimDuration().
scanGetAimDur() calls ScanGetAimDuration().
scanBeepAfterDecode(int bBeep) calls ScanSetBeepAfterGoodDecode(). bBeep= 1 to enable, bBeep= to disable.
scanGetBeepAfterDecode() calls ScanGetBeepAfterGoodDecode().
scanBiDi(int bBiDi) calls ScanSetBidirectionalRedundancy(). bBiDi = 1 to enable, bBiDi = to disable.
scanGetBiDi() calls ScanGetBidirectionalRedundancy().
scanLedOnTime(int time) calls ScanSetDecodeLedOnTime().
scanGetLedOnTime() calls ScanGetDecodeLedOnTime().
scanLaserOnTime(int time) calls ScanSetLaserOnTime().
scanGetLaserOnTime() calls ScanGetLaserOnTime().
scanCount(int n) calls ScanSetLinearCodeTypeSecurityLevel(). 1 = SECURITY_LEVEL1, 2 = SECURITY_LEVEL2, etc.
scanGetCount() calls ScanGetLinearCodeTypeSecurityLevel().
scanAngle(int angle) calls ScanSetAngle(). Use ANGLE_WIDE or ANGLE_NARROW
scanGetAngle() calls ScanGetAngle(). Returns ANGLE_WIDE or ANGLE_NARROW
scanTriggerMode(int mode) calls ScanSetTriggeringModes(). Use TRIG_LEVEL, TRIG_HOST, or TRIG_PULSE
scanGetTriggerMode() calls ScanGetTriggeringModes(). Returns TRIG_LEVEL, TRIG_HOST, or TRIG_PULSE
Note: The following Symbol APIs are currently not supported: Scan*BeepDuration, Scan*BeepFrequency, Scan*PrefixSuffixValues, Scan*ScanDataTransmissionFormat, Scan*TransmitCodeIdCharacter


Scanner Bar Code Functions
Function Name Description
scanEnableType(int type, int bEnable) calls ScanSetBarcodeEnabled(). type is one of values defined in scanner.h beginning with BAR_. bEnable = 1 to enable, bEnable = 0 to disable.
scanOptCode39(int bCheckDigitVerification, int bFullAscii) calls ScanSetCode39CheckDigitVerification() with the value of bCheckDigitVerification, and ScanSetCode39FullAscii() with the value of bFullAscii.
